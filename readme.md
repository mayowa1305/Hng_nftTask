# HNG_NFT_CLI



Generate CHIP-007 NFT JSON METADATA and Update Hash in a new CSV file.

## Getting started

install node.js on your computer.

1. ### Clone the repo or downlod it.
```sh
git clone 
```

2. ### Install all dependencies.
```sh
npm install
```

3. ### Start the program.

Simply run the `main.js` file.

```js
node main.js
```
